# WhiskeyWiz Project Setup

## Prerequisites
- Node.js 20.9.0 or later
- npm 9.x or later
- Git (optional)

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/nuwud/whiskeywiz.git
cd whiskeywiz
```

### 2. Run Setup Script
- On Windows: Run `project-setup.bat`
- On Mac/Linux: Run `./project-setup.sh`

### 3. Common Commands
```bash
# Start development server
nx serve

# Build project
nx build

# Run tests
nx test

# Lint project
nx lint
```

## Troubleshooting
- Ensure Node.js is correctly installed
- Check global package installations
- Verify Firebase configuration

## Project Structure
- Angular 17.x
- Nx Workspace
- Firebase Integration
- Web Components for Shopify
